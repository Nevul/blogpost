Esta es una intro de práctica de GIT y GITLAB

Bienvenidos al curso de Git y GitLab, donde exploraremos las herramientas esenciales para el control de versiones y la colaboración en proyectos de software. Aprenderemos a utilizar Git para el seguimiento de cambios y ramificaciones, mientras descubrimos cómo GitLab facilita la colaboración en equipo. Este curso proporcionará las habilidades fundamentales para gestionar eficientemente el desarrollo de proyectos, permitiéndoles maximizar la productividad y la coordinación. ¡Prepárense para sumergirse en el mundo de la gestión de versiones y potenciar su capacidad para trabajar de manera efectiva en equipo!

Como un extra, veremos superficialmente sobre Codespace, una herramienta que nos permite tener un entorno de desarrollo preconfigurado en una máquina virtual remota, de forma que podemos acceder a ella desde cualquier sitio y enseguida estar listos para empezar a trabajar, sin necesidad de configurar nada.

En el futuro estaremos creando ramas de estudio enfocadas hacia carreras independientes, conforme se vayan anexando más cursos.

En este momento estoy revisando el manejo de errores al realizar merge entre commits de diferentes branchs.
